# 🤖 项目风险监控 Agent
> 一个基于 Python + Trello API 的智能项目风险监控工具，自动追踪任务进度、识别延期风险并执行应对措施。

## 📌 项目背景
在研发项目管理中，任务延期是导致项目失败的首要原因之一。然而，大部分延期在发生初期都难以被及时发现，往往等到 deadline 临近才暴露，此时已错过最佳干预时机。

本项目旨在解决这一问题：设计一个轻量级的 AI Agent，自动监控项目任务进度，在风险发生早期发出预警并自动采取应对措施，将风险管理从"被动响应"转变为"主动预防"。

> 💡 本项目是我为求职 **项目管理岗** 而制作的个人作品，旨在展示我将技术能力与项目管理思维相结合的综合素质。

## 🎯 核心功能

| 功能模块 | 具体说明 |
|---------|---------|
| **自动任务创建** | 输入任务名称和预计天数，Agent 自动在 Trello 看板创建带截止日期的任务卡片 |
| **智能风险分级** | 基于任务剩余天数自动判定风险等级：逾期→严重风险，<2天→风险，其他→正常 |
| **可视化标签** | 自动为卡片添加标签：🔴 严重风险 / 🟡 风险 / 🟢 正常，项目状态一目了然 |
| **自动处理措施** | 对严重风险任务自动移回待办列表，并输出资源调配建议 |
| **项目报告生成** | 运行结束时自动生成任务统计报告（总数/各状态数量/风险数量） |

### Trello 看板效果
运行后，Trello 看板上的卡片会自动：
- 显示**截止日期**
- 带有**红色⚠️风险**或**绿色✅正常**标签
- 严重风险卡片自动从"进行中"移回"待办"


## 运行步骤
1. 注册 [Trello](https://trello.com) 账号
2. 在 Trello 中创建一个名为 **"无人机巡检系统开发"** 的看板
3. 在看板中创建三个列表：**待办**、**进行中**、**已完成**
4. 获取 Trello API 密钥

### 获取 Trello API 密钥
1. 登录 Trello 后访问：https://trello.com/power-ups/admin
2. 点击 **"创建新的Power-Up"**，填写：
   - 名称：`Project Risk Agent`
   - 工作空间：选择你的个人工作空间
   - 其他字段随意填写，**iframe连接器URL留空**
3. 创建成功后，页面会显示 **API Key**，复制保存
4. 在页面下方点击 **"生成Token"**，复制保存

```bash
